Detailed Description

The Event class holds the event record. The complete event record (in the form of a pointer to an instance of Event) is obtained via Sartre::generateEvent(). It is a lightweight class with only one member function and all data member are public. The class holds information about general event characteristics (Q2, W, t, x, etc) as well as a vector holding all particles involved in the event. Note that the instance of the current event is deleted when the next event is generated. In case you want to keep a certain event in your application you need to keep a copy.

Note that in the incoherent case (with nuclear breakup on) Sartre does not conserve momentum/energy. The reason is that the generator that takes care of the nuclear breakup (gemini++) does not generate photons but only baryons. The event record is thus not 100% complete. This might be fixed in future versions of Sartre.

Note that the products of the nuclear breakup in the incoherent case are only listed when activated through the runcard (enableNuclearBreakup = true) or programmatically (EventGeneratorSettings::setEnableNuclearBreakup(true)) . It is off by default. Momenta and energies for nuclei are always given as per nucleon.

Member Function Documentation

void list(ostream &os = cout) const

Prints the current event record to output stream os (default = cout). It is often useful to print few events of each run to see if everything is set correctly or for debugging purposes. See the examples above.

 

Data Member Documentation

unsigned long eventNumber

Number of the current event. Event numbering starts at 0.

double Q2

Q2 of the current event in GeV2.

double W

W of the current event in GeV2.

double t

t of the current event in GeV2.

double x

Bjorken x of the current event.

double s

Center-of-mass energy of the current event in GeV2. This is the same for all events in a run and is just recorded for convenience.

double y

Inelasticity, y, of the current event.

double xpom

xpomeron of the current event.

double beta

β of the current event. Can always be calculated via x/xpom but is recorded for convenience.

GammaPolarization polarization

Polarization of the virtual photon.

DiffractiveMode diffractiveMode

Records if the event is coherent or incoherent.

vector<Particle> particles

All particles stable or decayed in the current event. See the class Particle reference for details.
